SquaTTr

SquaTTr is an app that connects squatters with foreclosed bank-owned properties. Why rent, when you can squat? SquaTTr is not intended for actual use or public release -it's a joke app we created for a group project at UCLA Extension while learning JavaScript to draw attention to the affordable housing crisis in Los Angeles.
